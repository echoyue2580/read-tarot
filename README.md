# 🔮 悟东塔罗牌 Skill

基于悟东（Wudong）的塔罗牌知识库技能——78 张牌逐张含义、常用牌阵、解读方法论，支持查牌意、解牌阵、占卜解读。

## 快速安装

1. **克隆仓库**

```bash
git clone https://github.com/echoyue2580/read-tarot.git
```

2. **复制 skill 到你的悟东工作区**

```bash
cp -r read-tarot/.tron/skills/tarot <你的工作区路径>/.tron/skills/
```

3. **重启悟东会话**，skill 自动加载。

## 使用方式

安装后直接对话即可，无需特殊前缀：

| 你可以这样问 | 示例 |
|-------------|------|
| 抽牌占卜 | "帮我用三张牌阵看本周运势" |
| 查牌意 | "正义逆位什么意思？" |
| 解牌阵 | "过去 宝剑3，现在 隐士，未来 太阳 — 怎么解读？" |
| 选牌建议 | "新手买什么塔罗牌？" |
| 知识问答 | "凯尔特十字每个位置代表什么？" |
